# Seat avatars

Drop transparent portrait PNGs here, named by avatar id, to give seats premium
character art (the cyberpunk / mafia poker-boss style). Ids used by the table:

  boss.png  cyber.png  queen.png  ace.png  shark.png
  phantom.png  duchess.png  maverick.png  oracle.png

Square images (e.g. 256x256 or 512x512), face centered, work best — they render
in a circular glowing ring. Any missing file falls back to a generated monogram,
so you can add art incrementally. Assignment is deterministic per player id.
